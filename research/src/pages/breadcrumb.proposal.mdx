## Overview

Breadcrumbs are navigation land marks used to show a user's location within the site or app.

## Use Cases

A `<breadcrumb />` trail consists of a list of links to the parent pages of the current page in hierarchical order. It helps users find their place within a website or web application. Breadcrumbs are often placed horizontally before a page's main content.

#### Structure <a class="link" href="#structure" id="structure"></a>

- `<breadcrumb>` - The root element that that has the semantic value of navigation **[required]**
- `<ordered-list>` - The list containing items and dividers **[required]**
- `<list-item>` - The wrapper to items or divider content **[required]**

Proposed HTML:

```html
<nav>
  <ol>
    <li><a>Item 1</a></li>
    <li aria-hidden="true">/</li>
    <li><a>Item 2</a></li>
  </ol>
</nav>
```

#### Accessibility <a class="link" href="#accessibility" id="accessibility"></a>

The Breadcrumb is composed by:

- A `nav` equivalent wrapper that should always have `aria-label` or `aria-labelledby`
- A ordered `list`
- `li` items and dividers where the `li` represeting the item can contain anchor or other elements and the `li` represeting the divider should have `aria-hidden="true"`
- The link to the current page has aria-current set to page. If the element representing the current page is not a link, aria-current is optional.
