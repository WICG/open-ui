## Overview

Displays a placeholder reserving the place for the content that is being loaded or used to prototype layouts.

## Use Cases

The Skeleton component is primarily leveraged to reserve a place in the layout for a content that will be shown asynchronous avoiding elements moving to adapt to new dimensions.

#### Structure <a class="link" href="#structure" id="structure"></a>

- `<oui-skeleton>` - The root element that contains the lines or shapes and controls the internal animation **[required]**

### Content not allowed within the anatomy

Skeleton should be atomic **NOT** containing any interactive element inside
