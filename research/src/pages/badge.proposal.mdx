# Badge Component Specification

## Overview <a href="#overview" id="overview"></a>

_Badge_ is normally a label holding small amount of information such as value or status. Appears as some sort of notification with eye-catching appeal.

### Use Cases <a href="#use-cases" id="use-cases"></a>

The <badge /> control is primarily leveraged to indicate value of notifications or status. For example a badge in a avatar can indicate status as busy, available and other while a badge in a inbox can indicate the number of new messages.

## &lt;badge&gt; Properties <a href="#badge-properties" id="badge-properties"></a>

| Attribute Name | Type      | Default Value | Description                                                 |     |
| -------------- | --------- | ------------- | ----------------------------------------------------------- | --- |
| `value`        | `string`  | `""`          | Text value to be displayed by badge                         |     |
| `circular`     | `boolean` | `true`        | If the badge shape is circular                              |     |
| `color`        | `string`  | `red`         | Badge background color                                      |     |
| `ribbon`       | `string`  | `right`       | Badge can appear as a ribbon attaching itself to an element |     |
| `size`         | `string`  | `small`       | Badge can have different sizes                              |     |

#### &lt;badge&gt; Events <a href="#badge-events" id="badge-events"></a>

| Event Name | Detail Type | Bubbles | Composed | Cancellable | Dispatch Behavior                        |
| ---------- | ----------- | ------- | -------- | ----------- | ---------------------------------------- |
| `click`    | none        | `true`  | `true`   | `false`     | Fired when the user clicks on the badge. |
